Installation

Reanimated 2 is primarily build in C++ using Turbo Modules infrastructure which is not yet completely deployed in React Native (specifically on Android). Because of that the installation of new Reanimated requires additional steps apart from just adding a dependency to package.json.

As a consequence of the above the minimum suppoorted version of React Native is v0.62. Before you continue with the installation, make sure that you are running the supported version of React Native.
